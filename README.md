## âœ¨ Features

- Multi-account dashboard with status badges and Connect / Stop / Connect All
- Real Microsoft authentication for migrated Java accounts
- Per-account SOCKS5 proxy support
- Discord webhook alerts for joins, disconnects, deaths, kicks, and errors

- Simple JSON configs stored next to the executable

## ðŸ“¦ Download & Run

1. Go to the **Releases** tab on this repository.
2. Download the latest Windows build (for example `ConsoleClient-win64.zip`).
3. Extract the zip to a folder such as `C:\ConsoleClient`.
4. Run `ConsoleClient.exe`

Configuration (server host, version, join command, accounts, proxies, webhooks) is handled inside the app.
